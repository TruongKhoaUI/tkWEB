tkBOT privacy

Bot will save the server information such as the server name, server owner, server ID to the logs when invited or kicked.

The content such as warn data, commands enable/disable data on the guild will be save to the bot data to let the bot can be remember to use it for the bot's features and bot's logs.

Warns data is used for the warn command to update the warning count and the warning reason for the specifc user you choose. If you want to clear warn data, use the warndelete command to delete all of the warn for the specific member of the bot's data

The server count will be shared on the top.gg website to showing the server count of this bot that how many servers that using this bot, or you can show it on the bot using `botinfo` command.
