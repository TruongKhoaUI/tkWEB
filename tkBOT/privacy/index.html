tkBOT privacy

Data collection

Bot will collect the data such as server, username, or messages logs to the logs server that will visible to the owner of this bot only.

Bot will save the server information such as the server owner, server ID to the logs when invited or kicked.

Messages will be collected and encrypted the content such as the username, the time they have sent the message to the logs server that will be visible to the owner of this bot only.

The data collection will be removed manually by the bot owner if it doesn't need.
