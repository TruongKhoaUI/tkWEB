tkBOT privacy

Bot will collect the data such as server, username, or messages logs to the logs server that will visible to the owner of this bot only.

Bot will save the server information such as the server owner, server ID to the logs when invited or kicked.

The content such as warn data, commands enable/disable data on the guild will be save to the bot data to let the bot can be remember to use it for the bot's features and bot's logs.

Data removal you have to contact the bot owner to remove the data on specific guild from the bot by contact the owner on Discord.

The server count will be shared on the top.gg website or you can show it on the bot using `botinfo` command.
